#  Product Store App
#  How to Run Project
# install dependencies
npm install
# run project
cd product-store-app
npm run dev
A simple and modern React product store application built as a final assignment using React ecosystem tools.



# Project Description

This project is a small e-commerce application where users can:

- Browse products from an API
- Search, filter, and sort products
- View product details
- Add products to cart
- Manage cart items
- Change theme (dark/light)
- Change language (EN / FA / PS)
- Switch layout (Grid / List)


# State Management
- Context API + useReducer for:
  - Theme (Dark / Light)
  - Layout (Grid / List)
  - Language system

- Redux Toolkit for:
  - Shopping cart management
  - Add / Remove / Increase / Decrease items
  - Total price & quantity

- React Query for:
  - Fetching products from API
  - Loading & error handling
  - Cached data management
